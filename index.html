HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The 'src' element is the location or the source of the image
  • The 'alt' element is the text string that describes the image

CSS

  • A margin is space outside an element
  • Padding is the space inside the element

GitHub

  • GitHub is a version control methodology for development teams
  • It is used for content manegement, accessing repositories, tracking work, making and publishing changes to code
  • A repository is a project.
  • A branch is a workspace where you can make changes without effecting the main branch
  • Markdown files are content files. They're easily converted to HTML
  • A commit is saving a new version of the file you're working on
  • A Pull Request is asking for a changes on the main branch to incorporate the features or changes you've made on your own feature branch
  • An issue is a way to report new tasks or content fixes
  • Once an issue is made, you would create a branch to solve that particular issue, and then raise a pull request once you believe it's complete
